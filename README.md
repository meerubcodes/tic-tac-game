🎮 Tic Tac Toe Game

A simple Tic Tac Toe game built using HTML, CSS, and JavaScript.

🚀 Features
- 2 Player Game (X and O)
- Winner detection
- Draw detection
- Reset / New Game button

🛠️ Tech Used
- HTML
- CSS
- JavaScript

🎮 How to Play
Click boxes to play. First to make 3 in a row wins. If all boxes fill, it's a draw.
